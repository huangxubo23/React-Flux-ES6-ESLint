# react-todo
Another basic todo app in React and Flux

#### Run

```
npm install
npm start
```

#### Develop

Webpack auto-watches client assets.
Nodemon auto-watches server assets.

```
npm install
npm run dev
```

#### Build

Bundle the assets for production.

```
npm run build
```

#### Docker

Create and run the docker-compose environment

```
docker build -t todo .
docker-compose up
```
